AgenticFlict Documentation
==========================

**AgenticFlict** is a large-scale dataset of textual merge conflicts in AI
coding agent pull requests on GitHub, accompanied by the extraction pipeline
used to build it.

   :widths: 30 70
   :header-rows: 0

   * - **Repository**
